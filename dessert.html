Static Webpage for Pittsburgh's Happy Day Dessert Factory

About:

In the Spring of 2023, I created this front-end static webpage for Happy Day Dessert Factory, a local ice cream shop in Pittsburgh. The aim of this project was to create a more friendly user experience for customers than the existing website.

About my implementation:
  • Built from scratch using HTML/CSS and Javascript Libraries (Jquery, AOS)
  • Wireframed the Design of the website using Figma
  • Conducted user testing to maximize UX for all intended users

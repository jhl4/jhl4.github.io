Flashcards Command Line Interface

About Flashcards:

I have always found spaced repetition to be an excellent way to absorb information when I'm learning new things. This project allows me to simulate the use of the flashcards via a command line interface. All I need is a csv file with 1 column for the front side, and 1 column for the back side. Order, Repetitions, and invertCards are command line options I added to customize the flashcard learning experience.

My Implementation:

  • This program is written in Typescript.
  • I fully tested the program with Jest to achieve appropriate specification coverage.
  • I used the yargs library to parse user arguments and provide a clean user interface output on the terminal.
  • I adhered to encapsulation principles with the use of Typescript interfaces.
